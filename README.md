# Embed X Preview for Telegram

A Telegram bot designed to automatically embed and format X (formerly Twitter) posts for better visibility and interaction within Telegram group chats.

## Features
* **Auto-Embed:** Converts X links into rich media previews.
* **Pure Java:** Built with Java without heavy frameworks for maximum performance.

## Under Development 
* **Auto-Embed:** For Youtube Short, TikTok and Instagram

### Prerequisites
* Java 17 or higher
* Maven
* A Telegram Bot Token (from [@BotFather](https://t.me/botfather))
